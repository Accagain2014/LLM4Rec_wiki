# RecGPT Technical Report (RecGPT 技术报告)

**来源类型**：论文摘要
**作者**：Chao Yi, Dian Chen, Gaoyang Guo, Jiakai Tang 等（共54位作者）
**年份**：2025
**会议/期刊**：arXiv
**arXiv**：2507.22879

## 摘要

本文提出了一种以用户意图为核心的下一代推荐系统框架 RecGPT，旨在解决传统工业推荐系统过度依赖历史共现模式与日志拟合（log-fitting）所导致的兴趣窄化、信息茧房及长尾效应问题。RecGPT 将大语言模型（LLM）深度融入用户兴趣挖掘、商品召回与解释生成等关键环节，通过多阶段训练范式与人类-LLM协同评判机制，实现从“日志拟合”向“意图驱动”的范式转变。该系统已在淘宝 App 全面上线，在线实验验证了其在提升用户体验、商家曝光与平台转化方面的多赢效果。

## 主要贡献

1. **提出意图驱动的推荐新范式**：突破传统基于历史交互日志拟合的局限，首次将“用户意图”置于推荐流水线的中心，有效捕捉用户动态演化与潜在兴趣，缓解信息茧房与长尾分发难题。
2. **构建 LLM 与推荐系统深度融合的架构**：将通用大语言模型无缝接入兴趣挖掘、多路召回与可解释性生成三大核心模块，实现推荐流程的语义化与推理化升级。
3. **设计大规模领域对齐的多阶段训练策略**：提出“推理增强预对齐 + 自训练演进”的训练范式，并引入人类-LLM 协同评判系统，在保障推荐质量的同时实现模型在工业级数据上的高效对齐与持续进化。

## 方法

### 架构
RecGPT 采用以意图为中心的端到端推荐架构。系统底层以通用大语言模型为基座，上层划分为三大核心组件：
- **意图感知兴趣挖掘模块**：利用 LLM 的语义理解与推理能力，从用户历史行为、上下文及显式反馈中抽离深层意图，构建动态用户画像。
- **语义驱动召回模块**：将传统基于 ID 或向量的召回升级为基于意图语义的匹配，通过 LLM 生成查询表征或重排候选集，提升长尾与新颖内容的触达率。
- **可解释性生成模块**：在推荐结果输出阶段，由 LLM 自动生成个性化推荐理由与交互解释，增强推荐透明度与用户信任感。

### 关键技术
- **多阶段训练范式**：包含“推理增强预对齐（Reasoning-enhanced Pre-alignment）”与“自训练演进（Self-training Evolution）”两个阶段。预对齐阶段注入推荐领域的先验知识与推理逻辑，使 LLM 具备基础推荐能力；自训练阶段利用模型自身生成的高质量样本进行迭代优化，持续提升领域适应性。
- **人类-LLM 协同评判机制**：构建混合评估系统，结合人工标注的精准反馈与 LLM 自动化大规模评估，形成闭环奖励信号，指导模型在多样性、准确性与可解释性之间取得最优平衡。
- **工业级部署优化**：针对淘宝海量并发场景，采用模型压缩、缓存策略与异步推理管线，确保 LLM 在推荐链路中的低延迟与高吞吐。

## 实验结果

RecGPT 已在淘宝 App 完成全量部署，并通过大规模在线 A/B 测试验证其有效性。实验结果对比传统日志拟合基线系统，RecGPT 在多利益相关方指标上均实现显著提升：
- **用户侧**：内容多样性指标与用户满意度评分实现正向增长，信息茧房现象得到有效缓解。
- **商家与平台侧**：长尾商品曝光率与整体转化率（CVR）获得可观提升，流量分发更加均衡。
- **综合对比**：在线实验证实，意图驱动的 LLM 推荐范式在保持核心业务指标稳定增长的同时，显著优化了推荐系统的长期生态价值。（注：完整技术报告中详细披露了各核心指标的具体百分比增益数据与统计显著性检验结果，通常涵盖 CTR、CVR、多样性指数等维度的显著正向对比。）

## 局限性

- **推理延迟与计算成本**：尽管已进行工程优化，但 LLM 在推荐全链路中的引入仍带来较高的算力开销与推理延迟，对实时性要求极高的场景构成挑战。
- **意图建模的噪声与幻觉**：LLM 在复杂交互场景下可能产生意图误判或生成幻觉，需依赖更精细的协同评判与后处理机制进行纠偏。
- **冷启动与数据依赖**：多阶段训练仍依赖大规模高质量交互日志与人工标注数据，在极端冷启动或跨域场景下的泛化能力有待进一步验证。

## 与 LLM4Rec 的相关性

RecGPT 是 LLM4Rec 领域迈向工业级落地的标志性工作之一。它突破了以往 LLM 仅用于离线评估或辅助排序的局限，首次将大模型深度嵌入推荐系统的核心流水线（兴趣挖掘、召回、解释），并提出了可规模化训练的意图对齐范式。该工作为“大模型驱动推荐系统范式重构”提供了完整的工程实践路径，证明了 LLM 不仅能提升短期指标，更能通过意图建模优化长期生态健康度，对后续 LLM 在电商、内容分发等超大规模推荐场景中的架构设计与训练策略具有重要指导意义。
