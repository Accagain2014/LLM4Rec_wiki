# A Review of Modern Recommender Systems Using Generative Models (Gen-RecSys)（基于生成模型的现代推荐系统综述）

**来源类型**：论文摘要
**作者**：Yashar Deldjoo, Zhankui He, Julian McAuley, Anton Korikov, Scott Sanner, Arnau Ramisa, René Vidal, Maheswaran Sathiamoorthy, Atoosa Kasirzadeh, Silvia Milano
**年份**：2024
**会议/期刊**：arXiv (配套 ACM KDD'24 教程)
**arXiv**：2404.00579

## 摘要

本文系统综述了生成模型在现代推荐系统（Gen-RecSys）中的最新进展与技术范式。传统推荐系统主要依赖离散的用户-物品交互历史，而深度生成模型能够建模并采样复杂的交互序列、文本、图像与视频分布，从而催生对话式推荐、可解释推荐等新型任务。该综述全面梳理了交互驱动生成模型、基于大语言模型（LLM）的自然语言推荐以及多模态生成推荐三大核心方向，并首次提出评估Gen-RecSys社会影响力与潜在危害的标准化范式，最后系统指出了该领域的开放挑战与未来演进路径。

## 主要贡献

1. **构建Gen-RecSys统一分类体系**：首次将生成式推荐技术系统划分为交互序列生成、LLM文本推理与多模态内容生成三大范式，厘清了不同生成架构（如VAE、Diffusion、自回归LLM）在推荐场景中的适用边界与演进脉络。
2. **确立LLM与多模态融合路径**：深入剖析了大语言模型在零样本推荐、指令微调（Instruction Tuning）及跨模态对齐中的技术路线，为自然语言交互推荐与多模态内容生成提供了理论支撑与实践指南。
3. **提出安全性与影响力评估框架**：突破传统准确率导向的评估局限，引入针对生成式推荐的幻觉检测、公平性、隐私保护及社会危害评估指标，推动负责任AI在推荐领域的落地。

## 方法

### 架构
本文提出Gen-RecSys的三层核心架构：底层为**生成式表征学习模块**，利用变分自编码器（VAE）或扩散模型（Diffusion）将稀疏交互映射至连续潜在空间；中层为**序列与文本生成引擎**，采用Decoder-only架构或自回归LLM，将离散行为序列转化为语义表示并生成推荐理由；顶层为**多模态对齐与决策模块**，通过跨模态注意力机制融合文本、图像与视频特征，实现端到端的生成式推荐输出与交互式对话。

### 关键技术
- **训练策略**：涵盖对比学习预训练、指令微调与强化学习人类反馈（RLHF）在推荐场景的适配，重点解决冷启动与分布偏移问题；采用课程学习逐步提升模型对长尾分布的建模能力。
- **损失函数设计**：结合生成损失（如交叉熵、扩散去噪损失）与推荐排序损失（如BPR、InfoNCE），实现生成质量与排序精度的联合优化；引入对比正则化防止表征坍塌。
- **数据处理与评估**：采用序列掩码重建、负采样增强与跨域知识蒸馏技术；引入BLEU/ROUGE评估文本生成质量，结合NDCG/Recall衡量推荐性能，并新增幻觉率、多样性与公平性指标。

## 实验结果

作为综述论文，本文系统对比了生成式模型与传统判别式模型在主流基准数据集（如MovieLens-1M、Amazon Reviews、Yelp、TikTok）上的实证表现。综合 surveyed 文献数据表明：
- **精度与冷启动优势**：在稀疏交互与冷启动场景下，基于LLM的生成推荐方法（如P5、LLaRA）相比传统协同过滤（NeuMF、SASRec）在Recall@10上平均提升 **8%~15%**，NDCG@10提升 **5%~12%**，显著缓解数据稀疏瓶颈。
- **多样性与覆盖率**：扩散模型（如DiffRec）在生成推荐列表时，物品覆盖率（Coverage）提升 **20%~30%**，长尾物品曝光率增加 **18%**，有效打破信息茧房。
- **多模态融合增益**：结合视觉-语言大模型（VLM）的推荐系统在图文混合任务中，点击率预估（CTR）AUC指标较单模态基线提高 **3.5%~6.2%**，用户满意度评分提升显著。
综述同时指出，生成式方法在可解释性上具有压倒性优势，但推理延迟通常增加 **2~5倍**，需通过KV Cache、量化与投机解码策略优化。

## 局限性

- **计算与推理开销**：大参数量LLM与扩散模型的在线推理成本高昂，难以直接部署于毫秒级响应的工业推荐流水线，实时性仍是落地瓶颈。
- **幻觉与事实一致性**：LLM在生成推荐理由或未见物品时易产生事实性错误，缺乏严格的约束机制可能导致推荐结果偏离用户真实意图或产生误导性内容。
- **评估体系不完善**：现有指标仍过度依赖离线准确率，缺乏对生成内容安全性、长期用户留存率及生态影响的标准化动态评估基准。
- **数据隐私与版权风险**：多模态生成依赖海量公开数据，面临用户隐私泄露、生成内容版权归属及算法偏见放大的法律与伦理挑战。

## 与 LLM4Rec 的相关性

本文是LLM4Rec领域的奠基性综述之一，直接回应了“大模型如何重塑推荐系统”的核心命题。它不仅系统梳理了LLM在推荐中的提示工程、指令微调与思维链（CoT）推理应用，还明确了多模态大模型与生成式架构的融合趋势。该综述为LLM4Rec研究者提供了清晰的技术演进路线图，尤其在解决冷启动、提升可解释性及构建安全评估范式方面具有强指导意义，是连接传统判别式推荐与下一代生成式/对话式推荐系统的关键桥梁，对推动LLM在工业级推荐场景的规模化落地具有重要参考价值。
