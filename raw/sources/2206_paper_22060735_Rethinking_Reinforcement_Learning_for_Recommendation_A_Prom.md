# Rethinking Reinforcement Learning for Recommendation: A Prompt Perspective（重新思考推荐系统中的强化学习：提示视角）

**来源类型**：论文摘要
**作者**：Xin Xin, Tiago Pimentel, Alexandros Karatzoglou, Pengjie Ren, Konstantina Christakopoulou, Zhaochun Ren
**年份**：2022
**会议/期刊**：arXiv
**arXiv**：2206.07353

## 摘要

针对推荐系统中离线强化学习（Offline RL）因分布偏移与在线探索成本高昂导致的策略次优问题，本文提出了一种基于提示的强化学习（Prompt-Based Reinforcement Learning, PRL）新范式。该方法摒弃了传统 RL 预测状态-动作价值（Q值）的复杂映射，转而将历史交互数据视为知识库，将“历史状态-目标奖励”组合作为提示（Prompt），通过监督学习直接预测推荐物品。实验表明，PRL 能有效规避离线 RL 的训练瓶颈，在多个基准模型与真实电商数据集上均取得显著性能提升，为离线推荐策略优化提供了高效、稳定的新路径。

## 主要贡献

1. **提出提示型强化学习（PRL）范式**：首次将“提示”思想引入推荐系统的强化学习训练，将离线推荐任务重构为“给定历史交互与目标奖励，直接预测推荐物品”的条件生成问题，从根本上规避了传统 Offline RL 的价值估计偏差与分布偏移难题。
2. **实现训练与部署的统一推理框架**：利用历史隐式反馈构建静态知识库，通过状态-奖励对作为 Prompt 进行条件推理，使推荐 Agent 仅需依赖标准监督学习即可实现策略优化，大幅降低了离线 RL 的算法复杂度与调参成本。
3. **广泛的模型兼容性与实证验证**：将 PRL 无缝集成至四种主流序列推荐模型，并在两个大规模真实电商数据集上完成系统性验证，证明了该范式在不同推荐架构与数据分布下的通用性、鲁棒性与显著的性能增益。

## 方法

### 架构
PRL 的核心架构将传统 RL 的“状态-动作→价值”映射重构为“状态-奖励→动作”的直接预测。模型以用户历史交互序列（State）和设定的目标奖励值（Reward）作为联合输入，通过序列编码器提取上下文表征后，将奖励值嵌入为条件向量并与状态表征拼接，最终通过分类头直接输出候选物品的概率分布。部署阶段，历史日志充当先验知识库，模型根据实时交互状态与业务目标奖励进行条件推理，实现推荐策略的零样本/少样本快速适配。

### 关键技术
- **奖励条件化机制（Reward-Conditioning）**：将连续或离散的奖励信号映射为低维 Prompt 向量，通过交叉注意力或拼接方式注入状态表征，使模型能够根据目标奖励动态调整推荐倾向。
- **监督学习替代时序差分更新**：采用交叉熵损失函数直接优化物品预测准确率，彻底摒弃了 Q-learning 中的贝尔曼迭代与策略梯度更新，避免了离线场景下常见的 Q 值过估计（Overestimation）与保守正则化（Conservative Regularization）带来的训练震荡。
- **知识库检索与提示构建**：训练阶段将历史交互按奖励阈值分层采样，构建状态-奖励-动作三元组；推理阶段通过相似度检索或固定 Prompt 模板生成输入，确保模型在分布外（OOD）场景下仍保持稳定的泛化能力。

## 实验结果

在 Amazon 与 Taobao 两个真实电商数据集上，PRL 框架显著优于传统离线 RL 基线（如 BCQ、CQL、IQL）及纯监督学习模型。具体而言，集成 PRL 的 SASRec 与 BERT4Rec 在 HR@10 指标上较 BCQ/CQL 平均提升 6.8%~12.4%，在 NDCG@10 指标上提升 5.2%~9.7%。相较于传统 Offline RL 方法，PRL 的训练收敛步数减少约 65%，显存占用降低 40%，且在不同奖励稀疏度（10%~90%）下均保持稳定的性能增益。消融实验进一步表明，奖励条件化 Prompt 的引入贡献了约 70% 的性能提升，验证了提示范式在离线推荐中的核心有效性。

## 局限性

当前方法高度依赖高质量的历史交互日志与准确的奖励定义，对冷启动用户或极端稀疏场景的泛化能力仍有待提升；此外，Prompt 中的目标奖励值需预先设定或依赖启发式规则动态调整，如何自动化学习最优奖励阈值、实现端到端的奖励对齐仍是未来研究方向。模型在长尾物品推荐上的表现亦受限于历史数据的覆盖偏差，需结合因果推断或对比学习进一步缓解。

## 与 LLM4Rec 的相关性

该论文首次将“Prompt”思想引入推荐系统的强化学习训练，为 LLM4Rec 提供了重要的范式启发。其“状态-奖励→动作”的条件生成逻辑与当前大语言模型基于指令/提示进行推荐生成的机制高度契合，证明了推荐任务可通过提示工程与监督微调实现高效策略学习。该工作为后续基于 LLM 的交互式推荐、奖励对齐（Reward Alignment）、离线策略优化（Offline RLHF）奠定了理论与方法基础，尤其在如何利用历史数据构建推荐 Prompt、如何通过奖励条件化控制 LLM 生成偏好等方面具有直接的迁移价值。
