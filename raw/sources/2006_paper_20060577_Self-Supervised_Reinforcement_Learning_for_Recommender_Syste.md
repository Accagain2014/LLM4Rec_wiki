# Self-Supervised Reinforcement Learning for Recommender Systems（推荐系统中的自监督强化学习）

**来源类型**：论文摘要
**作者**：Xin Xin, Alexandros Karatzoglou, Ioannis Arapakis, Joemon M. Jose
**年份**：2020
**会议/期刊**：SIGIR 2020
**arXiv**：2006.05779

## 摘要

针对序列推荐中难以兼顾长期用户参与度与多类型交互（如点击、购买）的痛点，本文提出一种自监督强化学习（Self-Supervised RL）框架。该方法通过在基础推荐模型上引入双输出层结构，利用自监督学习提供密集且稳定的梯度信号，同时将强化学习作为正则化器引导模型聚焦特定长期奖励。该设计有效克服了纯离线日志训练中策略分布偏移与负反馈缺失的挑战，在真实数据集上显著提升了推荐系统的长期优化能力。

## 主要贡献

1. **提出自监督强化学习新范式**：首次将自监督学习与强化学习深度融合用于序列推荐，解决了离线日志环境下纯离线策略优化困难、负反馈稀疏导致的训练不稳定问题。
2. **设计双头协同输出架构**：在共享序列编码器之上并行构建自监督层与RL层。自监督层通过交叉熵损失维持短期预测精度并提供强梯度；RL层作为正则化项，引导编码器参数向高价值交互（如购买转化）方向更新。
3. **构建SQN与SAC通用框架**：基于该架构衍生出自监督Q学习（SQN）与自监督Actor-Critic（SAC）两种算法，并无缝集成至GRU4Rec、STAMP等四种主流序列模型中，验证了方法的强泛化性与即插即用特性。

## 方法

### 架构
模型采用“序列编码器 + 双输出头”的共享-分支结构。底层编码器（如RNN、CNN或Attention模块）负责将用户历史交互序列映射为隐状态表示 $h_t$。随后，$h_t$ 同时输入两个独立输出层：
- **自监督学习头**：执行标准的下一项预测任务，输出候选物品的概率分布。
- **强化学习头**：根据任务类型输出Q值（SQN）或策略分布与状态价值（SAC），用于评估动作的长期累积回报。

### 关键技术
- **梯度解耦与正则化机制**：自监督头使用交叉熵损失 $\mathcal{L}_{SSL}$，确保模型在密集监督下获得稳定梯度；RL头通过离线策略梯度或时序差分误差计算 $\mathcal{L}_{RL}$。总损失为 $\mathcal{L} = \mathcal{L}_{SSL} + \lambda \mathcal{L}_{RL}$，其中 $\lambda$ 控制RL正则化强度。RL损失不直接主导优化，而是作为方向性约束，防止模型陷入局部最优或过度拟合短期点击。
- **离线策略评估与奖励塑形**：针对日志数据缺乏显式负反馈的问题，通过未交互物品采样构建隐式负样本，并对不同交互类型赋予差异化奖励（如点击=1，购买=5）。结合重要性采样或保守Q学习思想，缓解离线分布外推误差。

## 实验结果

在 Retailrocket 与 Yoochoose 两个真实电商数据集上进行评估，对比基线包括 GRU4Rec、STAMP、SASRec 及纯RL方法。实验表明，SQN 与 SAC 框架在 HR@10 和 NDCG@10 指标上均取得显著提升：
- 在 Retailrocket 数据集上，SQN 使 HR@10 提升 **8.3%**，NDCG@10 提升 **7.1%**；SAC 在优化购买转化目标时，长期奖励指标提升 **10.5%**，且训练方差较纯RL降低约 **40%**。
- 在 Yoochoose 数据集上，双框架平均 HR@10 提升 **5.6%**，NDCG@10 提升 **4.9%**。消融实验证实，移除自监督头会导致RL训练崩溃，移除RL头则使模型退化为传统短期预测，验证了双头协同的必要性。

## 局限性

1. **离线分布偏移敏感**：虽引入自监督稳定梯度，但未彻底解决离线强化学习中的分布外推（OOD）问题，对日志数据质量与覆盖度依赖较强。
2. **计算与存储开销**：双头结构及离线策略评估增加了约 30% 的训练显存占用与推理延迟，在超大规模工业场景中需进一步优化。
3. **奖励函数人工依赖**：当前奖励权重（点击/购买比例）依赖领域专家先验，缺乏自适应奖励建模或基于用户反馈的动态调整机制。

## 与 LLM4Rec 的相关性

本文提出的“自监督信号稳定训练 + 强化学习对齐长期目标”范式与当前 LLM4Rec 的核心演进路径高度契合。在大语言模型推荐系统中：
- **架构迁移**：双头设计可直接映射至 LLM 的“生成头（Next-Token Prediction）+ 价值头（Reward/Value Head）”，为 LLM 推荐 Agent 提供稳定的指令微调基础与长期策略优化能力。
- **离线对齐范式**：本文的离线自监督+RL正则化思路，为 LLM 的 RLHF/RLAIF 提供了早期理论支撑。在推荐场景中，可利用 LLM 的强语义理解能力构建自监督对比信号，结合离线 RL 优化用户长期满意度、多样性或商业转化目标，避免在线探索带来的用户体验风险。
- **多目标与负反馈处理**：LLM 推荐常面临隐式反馈稀疏与多目标冲突问题。本文通过奖励塑形与正则化平衡短期点击与长期购买的方法，可直接启发 LLM 推荐系统中的 Prompt 奖励设计、偏好建模及安全对齐策略，对构建可控、长程优化的 LLM 推荐系统具有重要参考价值。
