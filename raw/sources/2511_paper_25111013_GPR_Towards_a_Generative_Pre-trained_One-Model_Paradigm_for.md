# GPR: Towards a Generative Pre-trained One-Model Paradigm for Large-Scale Advertising Recommendation (GPR：面向大规模广告推荐的生成式预训练单模型范式)

**来源类型**：论文摘要
**作者**：Jun Zhang, Yi Li, Yue Liu 等（共22位作者）
**年份**：2025
**会议/期刊**：arXiv
**arXiv**：2511.10138

## 摘要
针对传统多阶段广告推荐系统存在的目标不一致与误差累积问题，以及现有统一生成式推荐模型难以满足工业级应用需求的现状，本文提出 GPR（Generative Pre-trained Recommender）。该框架首次将大规模广告推荐重构为端到端的生成式任务，通过统一的语义表示、异构分层解码架构与多阶段联合训练策略，实现了兴趣建模、价值对齐与策略优化的全局统一，并在腾讯视频号广告系统中成功全量部署，显著提升了核心商业指标。

## 主要贡献
1. **提出首个端到端生成式广告推荐单模型范式**：打破传统“召回-粗排-精排”多阶段级联架构的局限，将广告推荐统一建模为序列生成任务，从根本上解决多阶段目标不一致与误差传播问题，实现全局最优。
2. **设计面向广告场景的统一输入与分词机制**：构建共享的多层级语义 ID 空间，将广告内容与自然内容映射至同一表示体系，有效增强异构数据间的语义对齐能力与建模一致性。
3. **开发异构分层解码器（HHD）与多阶段联合训练策略**：通过双解码器架构解耦用户意图建模与广告生成过程，在保障强大建模能力的同时兼顾训练效率与推理灵活性；结合多 Token 预测（MTP）、价值感知微调与层级增强策略优化（HEPO），形成完整的生成式推荐训练管线。

## 方法
### 架构
GPR 采用“统一编码器 + 异构分层解码器（HHD）”的端到端生成式架构。整体模型摒弃了传统推荐系统中独立的候选集生成与排序模块，直接将用户历史行为序列、上下文特征与候选广告池映射为统一的离散 Token 序列。HHD 作为核心生成模块，采用双解码器设计：上层解码器专注于深层用户意图与长期兴趣表征的提取，下层解码器负责基于意图表征自回归地生成目标广告 ID。该架构实现了意图理解与内容生成的显式解耦，既保留了生成式模型的灵活性，又避免了单一解码器在复杂广告场景下的训练不稳定与梯度冲突问题。

### 关键技术
- **统一语义 ID 分词（Unified Tokenization）**：针对广告与自然内容的异构性，设计多层级语义 ID 映射方案。通过层次化聚类与语义对齐技术，将高维稀疏的广告特征压缩至共享的离散 ID 空间，降低词汇表规模的同时保留细粒度商业属性（如类目、出价、转化倾向）。
- **多阶段联合训练策略**：
  1. **多 Token 预测（MTP）**：在预训练阶段引入并行多步预测机制，加速序列建模收敛，提升模型对长程用户行为序列的捕捉能力。
  2. **价值感知微调（Value-Aware Fine-Tuning）**：在监督微调阶段引入广告商业价值（如 eCPM、转化率预估）作为辅助监督信号，使生成目标与平台收益最大化对齐。
  3. **层级增强策略优化（HEPO）**：结合强化学习思想，设计层级奖励机制，在推理阶段动态优化生成策略，平衡探索与利用，确保生成结果兼顾用户兴趣与商业价值。

## 实验结果
GPR 已在腾讯视频号（Weixin Channels）广告推荐系统中完成全量线上部署。实验表明，相较于传统多阶段级联推荐架构，GPR 在核心业务指标上取得显著突破：
- **GMV（商品交易总额）**：实现显著提升，验证了生成式范式在直接驱动商业转化方面的有效性。
- **CTCVR（点击后转化率）**：获得大幅优化，表明模型在精准匹配用户意图与高价值广告方面具备更强能力。
（注：摘要中明确指出各项指标均实现显著提升，具体相对增益百分比与 A/B 测试置信区间详见论文正文实验章节。工业级部署验证了该架构在超大规模流量下的稳定性与低延迟推理能力。）

## 局限性
尽管 GPR 在工业场景中表现优异，但生成式推荐架构仍面临以下潜在挑战：1）自回归生成机制在超大规模候选集下的推理延迟仍需进一步优化，以满足广告系统毫秒级响应要求；2）多层级语义 ID 的构建高度依赖高质量的数据聚类与对齐算法，冷启动广告或长尾内容的语义映射可能存在偏差；3）联合训练策略涉及多目标优化，超参数调优与训练稳定性控制复杂度较高。未来工作可探索非自回归生成加速、动态词表扩展及更高效的在线学习机制。

## 与 LLM4Rec 的相关性
本文是 LLM4Rec（大语言模型赋能推荐系统）范式在工业级广告推荐领域的重要实践。GPR 将推荐任务彻底重构为“预训练+生成”的端到端流程，与当前 LLM4Rec 的核心思想高度契合：1）**统一建模范式**：借鉴 LLM 的 Next-Token Prediction 思想，将离散推荐 ID 视为语言 Token，实现跨模态、跨任务的统一表征；2）**价值对齐技术迁移**：引入类似 RLHF 的 HEPO 算法与价值感知微调，将大模型领域的“人类反馈对齐”理念转化为“商业价值对齐”，为 LLM 在推荐系统中的商业化落地提供了可复用的技术路径；3）**工业级可扩展性验证**：证明了生成式架构在超大规模、高并发广告场景下的可行性，为后续基于 LLM 的端到端推荐系统（如 Generative Retrieval、LLM-based Ranking）提供了重要的架构参考与工程经验。
